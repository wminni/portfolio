MiniCTF – Web Challenge

SQL Injection & Command Injection

Try the challenge

Launch MiniCTF Challenge

Direct link: https://mini-ctf.onrender.com/

This is a lightweight, beginner-friendly project built as an introduction to CTF challenge development and practical web exploitation.

This project is a self-built Capture the Flag (CTF) web challenge designed to demonstrate two common web application vulnerabilities: SQL Injection and Command Injection.

The challenge simulates a vulnerable login page that can be bypassed with crafted input, followed by an interface that accepts shell commands. The user can execute user-level OS commands such as `whoami` through unsanitized input and retrieve a base64-encoded flag, which must be decoded and submitted to complete the challenge.

The project includes a flag verification mechanism and a final summary explaining how the vulnerabilities were exploited and how they could be prevented. It reflects both offensive techniques and basic secure coding principles.

Technologies used
  • Python (Flask)
  • HTML and CSS
  • Python subprocess module
  • Render for deployment

This was my first self-designed CTF challenge, created to explore offensive security from the perspective of a challenge builder.
